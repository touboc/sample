## 通过Cli下载youtube-video

利用yt-dlp这个命令工具来下载视频。
[1]从github网站上下载youtube-dl这软件包
https://github.com/yt-dlp

[2]保存到path路径下
`C:\My\bin\yt-dlp-2026.02.21`

[3]做成自定义脚本(yt.bat)
```bash
@echo off

set video_url=%1
set file_name=%2
set lang=%3

cd /d C:\My\bin\yt-dlp-2026.02.21

yt-dlp "%video_url%" -o "C:\Users\user\Downloads\%file_name%" -S "res:720,ext:mp4" --write-auto-subs --sub-langs "%lang%"
```
[4]通过执行这个自定义脚本，来取得youtube视频

```bat
yt "https://www.youtube.com/shorts/RFYnaiqvGL0?feature=share" new-filename ja
```
